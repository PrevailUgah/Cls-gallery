# 🖼️ The Geng Gallery

A modern, interactive photo gallery web application built with HTML, CSS, and JavaScript. Showcasing a responsive and user-friendly interface for browsing and interacting with images.

**[🚀 View Live Demo](https://prevailugah.github.io/Cls-gallery/)**

---

## 📋 About

The Geng Gallery is a fully responsive photo gallery application that demonstrates modern web development practices. Users can browse through a curated collection of images with interactive features including download, like, and share functionality.

---

## ✨ Key Features

- **Responsive Design** – Seamlessly adapts to desktop, tablet, and mobile devices
- **Interactive Image Cards** – Hover effects with action buttons (download, like, share)
- **Bootstrap Integration** – Built on Bootstrap 5 for a professional, polished look
- **Smooth Animations** – Animated typing effect on homepage using Typed.js
- **Icon Library** – FontAwesome integration for beautiful, scalable icons
- **Modern Typography** – Custom Google Fonts (Montserrat) for enhanced aesthetics
- **Clean Code Structure** – Well-organized folder architecture for easy maintenance
